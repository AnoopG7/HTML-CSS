What is HTML

HTML stands for HyperText Markup Language. It is the standard language used to create and design the structure of web pages. HTML elements are represented by <tags>, which are used to define content such as headings, paragraphs, links, images, and more. HTML works alongside CSS and JavaScript to create complete, interactive websites.

What is a semantic tag?

A semantic tag is an HTML element that conveys meaning about the content it contains, helping both browsers and developers understand the content's structure. Examples: <article>, <section>, <header>, <footer>.

Difference between id and class?

The id attribute is used to identify a unique element, whereas the class attribute is used to apply styles or group elements together. An id should be unique within the page, while a class can be reused across multiple elements.

What is the <summary> element?

The <summary> element defines a visible heading for a <details> element. It's used to provide a clickable label that toggles the visibility of the content inside the <details> element.

What is CSS?

CSS (Cascading Style Sheets) is a style sheet language used to describe the presentation of a document written in HTML or XML. It controls the layout, colors, fonts, and overall appearance of the web page.
